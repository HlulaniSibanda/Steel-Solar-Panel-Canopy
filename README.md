## Project Overview

This project presents the complete structural design, BIM modelling, and structural analysis of a steel solar panel canopy intended for parking and outdoor shading applications.

The objective of this project is to demonstrate professional structural engineering capability for US-based steel, solar, and light commercial structural projects.

The model includes:

* Steel structural framing system
* Solar canopy support structure
* Bracing system
* Connection detailing
* Structural analysis using Autodesk Robot Structural Analysis
* Permit-style structural drawing sheets

This project is for portfolio and demonstration purposes only.

---

## Project Objectives

* Design a realistic steel solar canopy structure
* Demonstrate professional Revit structural workflow
* Perform structural analysis using Robot Structural Analysis
* Show clear load paths and structural behaviour
* Produce professional structural drawing sheets
* Demonstrate US-facing structural engineering workflow
